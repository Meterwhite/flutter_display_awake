## 1.0.0

- Initial stable release.
- Added `keepScreenOn()` to keep the display awake.
- Added `keepScreenOff()` to restore default lock/sleep behavior.
- Added `isKeptOn()` API across Android, iOS, macOS, Linux, Windows, and Web.
- Added example app demonstrating plugin usage and live state updates via `isKeptOn()`.
- Improved package documentation and metadata for pub.dev publication.
